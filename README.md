# CodemaoHyh 编程猫后援会 官网项目

## 必读内容

### 安装配置插件
```
npm install
```

### 运行
```
npm run serve
```

### 打包成html 减小项目大小
```
npm run build
```

### 检查
```
npm run lint
```

### 自定义配置
请参阅[配置参考](https://cli.vuejs.org/config/)
